Overview

Knockout-jQueryUI is a collection of Knockout bindings for jQuery UI widgets. Examples are on the bindings' API page. Source is on Github. This project was inspired by Ryan Niemeyer's Knockout-Kendo.js

Highlights

  • Small size
  • Isolated binding parameters.
  • Proper disposal of the bindings.
  • Synchronous operation. No setTimeout() when applying the bindings.
  • The bindings are protected against multiple initializations.
