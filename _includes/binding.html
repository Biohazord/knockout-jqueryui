Parameters

There is a corresponding parameter for each of the widget's options. These can be set to either normal or observable properties of the viewmodel.

There is a corresponding parameter for each of the widget's events. These can be set to functions of the viewmodel. The bindings invoke these functions in the context of the window object.

refreshOn

If set to one of the viewmodel's observables, then every time that observable changes, the widget is refreshed.

widget

If set to one of the viewmodel's observables, then the binding will write the widget instance into it.
