Parameters

There is a corresponding parameter for each of the widget's options. These can be bound either to normal or to observable properties of the viewmodel.

There is a corresponding parameter for each of the widget's events. These can be bound to functions of the viewmodel. The bindings invoke these functions in the context of the window object.

{% if page.hasRefresh == true %}

refreshOn

If bound to one of the viewmodel's observables, then every time that observable changes, the widget is refreshed (it's refresh method is invoked).

{% endif %}

widget

If bound to one of the viewmodel's writeable observables, then the binding will write the widget instance into it.
